\sol{1. 내용 불일치}{③}
안내문에 따르면 무료 시식(free sample tastings)은 \textbf{오후 2시부터 3시까지} 1시간 동안만 제공된다(\textit{between 2 p.m.\ and 3 p.m.}). 그러나 ③은 ``행사 전체 시간 동안 시식이 가능하다''고 서술하고 있어 본문 내용과 \textbf{일치하지 않는다}.\par
\vspace{2pt}
\textbf{오답 확인(모두 본문과 일치):}\par
\hspace{1em}① \textit{yearly basis} = \textit{annual} (해마다 열리는 연례 행사)\par
\hspace{1em}② \textit{at any point during the festival} = \textit{throughout the festival} (행사 내내 라이브 음악 공연)\par
\hspace{1em}④ \textit{do not need to pay anything to enter} = \textit{no entry fee} (입장 무료)\par
\hspace{1em}⑤ \textit{hands-on activity for younger visitors} = \textit{Cookie-making classes for children} (어린이 대상 체험 활동)

\vspace{8pt}{\sffamily\bfseries 2. 서술형}\par\vspace{4pt}

(1) \fbox{모범답안} \ \textbf{주차는 무료로 이용 가능하다.}\par
\vspace{1pt}\hspace{1.5em}— 근거 문장: \textit{Parking is available for free.} \quad 채점 기준: ``무료 주차'' 내용이 포함되면 정답.

\vspace{6pt}
(2) \fbox{모범답안} \ \textbf{between 2 p.m.\ and 3 p.m.}\par
\vspace{1pt}\hspace{1.5em}— 근거 문장: \textit{Free sample tastings between 2 p.m.\ and 3 p.m.} \quad \textit{2 p.m.\ to 3 p.m.}도 정답으로 인정.

\vspace{8pt}
\sol{3. 글의 목적}{②}
안내문은 Food Truck Festival의 날짜, 장소, 주요 행사, 입장료와 주차 정보를 소개한다. 따라서 축제와 주요 볼거리를 안내하는 ②가 정답이다.

\vspace{6pt}{\sffamily\bfseries 4. 서술형}\par
\fbox{모범답안} \quad \textbf{10 hours}\par
오전 11시부터 오후 9시까지이므로 총 10시간이다. 숫자만 쓴 ``10''도 정답으로 인정.
