Abstract

이 페이지는 방학 동안 읽을 VLA, meta-learning, in-context learning, physical AI 논문들의 읽기 순서와 읽기 노트를 모아두는 곳이다. 각 노트는 PDF를 옆에 둔 상태에서 page, section, figure, table, equation 기준으로 따라가게 만든다.

첫 번째 읽기 노트는 What Can Transformers Learn In-Context? 로 시작한다. 이 논문은 "Transformer가 context 안에서 실제로 어떤 학습 알고리즘을 흉내낼 수 있는가"를 가장 깨끗한 toy setting에서 보여준다. 이 렌즈를 잡고 나면 trajectory를 prompt처럼 다루는 sequence-modeling RL 논문들이 훨씬 잘 읽힌다.

3. How to Use

  1. 로컬 PDF 폴더에서 해당 PDF를 왼쪽에 열고, 이 사이트의 읽기 노트를 오른쪽에 둔다.
  2. 먼저 읽기 노트의 "읽는 이유"와 "배경지식"만 보고 PDF 본문으로 들어간다.
  3. 본문에서는 페이지 가이드를 따라가되, figure/table 해설은 해당 그림을 실제로 볼 때만 읽는다.
  4. 마지막에는 "비판적으로 볼 점"과 "다음 논문 연결"만 다시 훑고 다음 논문으로 넘어간다.
