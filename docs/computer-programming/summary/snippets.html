① 상속 + 오버라이딩 (추상클래스)

abstract class Shape {
    abstract double area();        // 추상 메서드
}
class Circle extends Shape {
    double r;
    Circle(double r){ this.r = r; }
    @Override
    double area(){ return 3.14*r*r; }
}

② 인터페이스 다중 구현

interface Flyable  { void fly(); }
interface Swimmable { void swim(); }

class Duck implements Flyable, Swimmable {
    public void fly() { System.out.println("fly"); }
    public void swim(){ System.out.println("swim"); }
}
클래스는 단일 상속, 인터페이스는 다중 구현 가능

③ Comparable (클래스 내부, 인자 1개)

class Item implements Comparable<Item> {
    int price;
    @Override
    public int compareTo(Item o){
        return this.price - o.price;   // 오름차순
    }
}
Collections.sort(list);   // 그냥 sort
반환: 음수→앞, 0→같음, 양수→뒤 · 내림차순 o.price - this.price

④ Comparator (외부 클래스, 인자 2개)

class PriceComp implements Comparator<Item> {
    @Override
    public int compare(Item a, Item b){
        return a.price - b.price;      // 오름차순
    }
}
Collections.sort(list, new PriceComp());
⚠️ implements(extends 아님) · 반환타입 int · @Override · 2차정렬: if(a.x!=b.x) return a.x-b.x; return a.y-b.y;

⑤ 커스텀 예외 (자주 출제)

class MyException extends Exception {   // Checked
    MyException(String msg){
        super(msg);                    // ⚠️ println 아님!
    }
}
void foo(int x) throws MyException {    // ⚠️ Checked면 throws
    if(x < 0)
        throw new MyException("음수: " + x);  // ⚠️ throw new
}
// 호출
try { foo(-1); }
catch(MyException e){
    System.out.println(e.getMessage());
}
⚠️ super(msg) 안 하면 getMessage() = null · RuntimeException 상속하면 Unchecked(throws 불필요)

⑥ 파일 읽기 (FileReader / Scanner)

// FileReader: read()→int, -1이면 EOF
FileReader fr = null;                  // ⚠️ try 밖 선언
try {
    fr = new FileReader("in.txt");
    int c;
    while((c = fr.read()) != -1)
        System.out.print((char) c);    // ⚠️ (char) 캐스팅
} catch(IOException e){ e.printStackTrace(); }
finally { /* fr.close(); */ }

// Scanner: nextLine() (readLine 아님!)
Scanner sc = new Scanner(new File("in.txt"));
while(sc.hasNext())
    System.out.println(sc.nextLine()); // ⚠️ nextLine
⚠️ Scanner=nextLine() / BufferedReader=readLine() · 자원은 try 밖에 선언해야 finally서 close

⑦ 파일 쓰기 (FileWriter) + finally

static void write(String text) {
    FileWriter fw = null;              // try 밖
    try {
        fw = new FileWriter("out.txt");
        fw.write(text);
        fw.close();
    } catch(IOException e){
        System.out.println(e.getMessage());
    } finally {
        System.out.println("done");    // 항상 실행
    }
}

⑧ 제네릭 메서드 + bounded type

static <T extends Comparable<T>> T larger(T a, T b){
    return a.compareTo(b) >= 0 ? a : b;
}
// 사용: larger(3, 8) → 8 / larger("a","z") → z
⚠️ 순서: [제어자] <T extends X> [반환타입] 메서드명 — <T>는 반환타입 앞!

⑪ C++ 클래스 + 초기화 리스트 + this

class Rect {
    int w, h;
public:
    Rect(int w, int h) : w(w), h(h) {}   // 초기화 리스트
    int area(){ return this->w * this->h; } // this는 포인터
    ~Rect(){ cout << "destroyed"; }       // 소멸자
};
초기화 리스트는 본문보다 먼저 실행 · this는 Java와 달리 포인터(this->)

⑫ C++ 복사 생성자 (deep copy) + 소멸자

class Buffer {
    int* data; int size;
public:
    Buffer(int n) : size(n) { data = new int[n]; }
    // 깊은 복사: 새 메모리 할당 후 복사
    Buffer(const Buffer& o) : size(o.size) {
        data = new int[size];
        for(int i=0;i<size;i++) data[i] = o.data[i];
    }
    ~Buffer(){ delete[] data; }          // ⚠️ delete[]
};
포인터 멤버 있으면 깊은 복사 필수 (얕은 복사 → double free) · 배열은 delete[]

⑬ C++ virtual / override / 가상소멸자 ★Lab

class Animal {
public:
    virtual void eat(){ cout << "..."; }  // virtual 필수
    virtual void cry() = 0;                // 순수 가상(추상)
    virtual ~Animal(){}                    // 가상 소멸자
};
class Cow : public Animal {                // public 상속
public:
    void eat() override { cout << "grass"; }
    void cry() override { cout << "moo"; }
};
Animal* a = new Cow();
a->eat();        // virtual이라 Cow::eat (동적)
delete a;        // 가상소멸자 → Cow→Animal 순 소멸
⚠️ C++은 virtual 있어야 동적 바인딩 · base 포인터 delete엔 가상소멸자 필수

⑭ C++ 2D 배열 (이중 포인터) ★Lab

int N = 3;
int** m = new int*[N];              // 행 포인터 배열
for(int i=0;i<N;i++){
    m[i] = new int[N];              // 각 행
    for(int j=0;j<N;j++) m[i][j] = 0;
}
// 해제 (역순)
for(int i=0;i<N;i++) delete[] m[i]; // 행 먼저
delete[] m;                          // 포인터 배열

⑮ C++ swap (포인터 / 레퍼런스)

// 포인터 방식
void swap(int* a, int* b){
    int t = *a; *a = *b; *b = t;
}
swap(&x, &y);   // 호출 시 & 필요

// 레퍼런스 방식 (더 깔끔)
void swap(int& a, int& b){
    int t = a; a = b; b = t;
}
swap(x, y);     // & 불필요

⚠️ 코드 수정 단골 (에러 찾기)

arr.size()   → arr.length      // 배열은 length
throw MyEx() → throw new MyEx() // new 빠짐
public write(...) → public void write(...) // 반환타입
static에서 인스턴스 변수 접근  → 컴파일 에러

catch(Exception) 먼저 → 하위 catch unreachable 에러
double→int 대입 → (int) 명시 캐스트 필요
