타입 시스템 · 캐스팅 · String · 연산자

main() 메서드

public static void main(String[] args) — 모든 Java 프로그램의 진입점.
· public: 어디서나 접근 가능
· static: 클래스 소속 (객체 생성 없이 호출 가능)
· void: 반환값 없음
· String[] args: 커맨드라인 인수
public 클래스 이름은 반드시 파일명과 일치해야 한다.

타입크기범위 / 특징리터럴
byte8 bits-128 ~ 127
short16 bits-32768 ~ 32767
int32 bits-2^31 ~ 2^31-1기본 정수 리터럴

타입 캐스팅

Widening (확장, 자동): 작은 타입 → 큰 타입
byte → short → int → long → float → double

Narrowing (축소, 명시적): 큰 타입 → 작은 타입, 반드시 (type) 캐스트 필요 
double d = 9.99;
int i = (int) d;   // 9 — 소수점 버림 (반올림 아님!)

String

String은 기본 타입이 아닌 참조 타입 (Reference type). 대문자 시작.
불변 (Immutable): 한번 생성된 String 객체의 내용은 변경 불가.
생성 방식저장 위치== 비교
String s = "Hello";String Pool (공유)같은 리터럴이면 true
String s = new String("Hello");Heap (새 객체)항상 false
String 내용 비교는 반드시 .equals() 사용!
String a = new String("Apple");
String b = new String("Apple");
a == b;        // false — 주소 비교
a.equals(b);   // true  — 내용 비교

연산자

산술 연산

정수 나눗셈: 5 / 2 = 2 (소수점 버림)
5 / 0 → ArithmeticException (정수 / 0은 런타임 에러) 
int x = 1, y = 1;
x + ++y;   // 3 — prefix: y가 먼저 2로 증가 후 덧셈
x + y++;   // 2 — postfix: y=1로 덧셈 후 2로 증가

논리 연산 — Short-Circuit

&&: 좌변이 false면 우변 평가 안 함
||: 좌변이 true면 우변 평가 안 함

연산자 우선순위 (높음 → 낮음)

String + 연산

System.out.println("답은: " + 2 + 3 + "!");  // 답은: 23!
System.out.println(2 + 3 + " is the answer"); // 5 is the answer
+ 연산은 왼쪽부터 평가. String이 먼저 오면 이후는 모두 문자열 연결.
