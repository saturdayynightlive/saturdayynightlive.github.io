Input / Output Lecture 10

Stream · Reader/Writer · Scanner

Java I/O 구조

분류클래스단위
StreamInputStream / OutputStream바이트 (byte)
Reader/WriterReader / Writer문자 (char, 2bytes, Unicode)
텍스트 파일은 FileReader/Writer, 바이너리 파일은 FileInputStream/OutputStream 사용.

InputStream 계열

InputStream (추상): 바이트 단위 입력의 최상위 클래스.
핵심 메서드: int read() — 1바이트 읽기, 끝이면 -1 반환

ByteArrayInputStream — 메모리에서 읽기

byte[] buf = { 35, 36 };
ByteArrayInputStream is = new ByteArrayInputStream(buf);
int k;
while ((k = is.read()) != -1) {
    System.out.println("ASCII:" + k + "; Char: " + (char) k);
}
// ASCII:35; Char: #
// ASCII:36; Char: $
is.close();

FileInputStream — 파일에서 읽기

FileInputStream fis = new FileInputStream("say_hello.txt");
int c;
while ((c = fis.read()) != -1) {
    System.out.print((char) c);
}
fis.close();
FileInputStream은 FileNotFoundException / IOException을 반드시 처리해야 함.

OutputStream 계열

OutputStream (추상): 바이트 단위 출력의 최상위 클래스.
핵심 메서드: write(int i), flush(), close()

FileOutputStream — 파일에 쓰기

FileOutputStream fos = new FileOutputStream("file.txt");
fos.write(65);  // 'A' 씀
fos.flush();
fos.close();

BufferedOutputStream — 버퍼링으로 효율 향상

FileOutputStream fos = new FileOutputStream("file.txt");
BufferedOutputStream bos = new BufferedOutputStream(fos);
bos.write("Hello World".getBytes());
bos.flush();
bos.close();
fos.close();

FileWriter / FileReader (문자 기반)

FileWriter — 파일에 문자 쓰기

FileWriter fw = new FileWriter("output.txt");       // 덮어쓰기
FileWriter fw = new FileWriter("output.txt", true); // 추가(append) 모드
fw.write("Hello");
fw.close();

FileReader — 파일에서 문자 읽기

FileReader fr = new FileReader(new File("data/hello.txt"));
int c;
while ((c = fr.read()) != -1) {
    System.out.print((char) c);
}

Scanner — 고수준 파일 읽기

Scanner input = new Scanner(new File("data/hello.txt"));
while (input.hasNext()) {
    System.out.println(input.nextLine());
}

파일 경로

상대 경로: "say_hello.txt" → 프로젝트 루트 기준 (IntelliJ)
절대 경로: "c:/java_program/say_hello.txt"
