Java Basics II Lecture 03

배열 · 조건문 · 반복문 · 함수

배열 (Arrays)

배열: 동일한 타입의 값을 연속된 메모리에 저장하는 자료구조.
인덱스는 0부터 시작, 마지막 인덱스는 length - 1. 
// 선언 방법 3가지
int[] arr = new int[3];             // 빈 배열 (0으로 초기화)
int[] arr = new int[] {1, 2, 3};   // 값 지정
int[] arr = {1, 2, 3};              // 단축 표기

arr.length;  // 배열 크기 (메서드 아님, 필드)

다차원 배열

int[][] matrix = {{1, 2}, {3, 4, 5}};
matrix[1][2];  // 5

조건문

if / else if / else

드모르간 법칙:
(!A) && (!B) == !(A || B)
(!A) || (!B) == !(A && B)

삼항 연산자 (Ternary)

String result = (a > b) ? "a가 큼" : "b가 큼";

switch 문

switch (expression) {
    case x:
        // ...
        break;   // break 없으면 다음 case로 fall-through!
    default:
        // ...
}
switch에서 사용 가능한 타입: byte, short, char, int, String, Enum, 래퍼 타입(Integer 등)
break를 빠트리면 다음 case로 실행이 이어진다 (fall-through)!

반복문

종류특징주의
while조건이 true인 동안 반복
do-while최소 1회 실행 후 조건 확인끝에 ; 필수!
for초기화; 조건; 증감
for-each배열/컬렉션 순회인덱스 접근 불가
// do-while: 세미콜론 주의
do {
    // ...
} while (condition);  // ← 세미콜론 필수

// for-each
for (String car : cars) {
    System.out.println(car);
}

break / continue

break: 반복문 전체를 즉시 종료.
continue: 현재 반복만 건너뛰고 다음 반복으로 이동.

함수 (Methods)

함수: 특정 작업을 수행하는 코드 블록. 이름으로 호출하여 재사용.
· 모듈화: 코드를 작은 단위로 분리
· 재사용성: 한 번 정의, 여러 번 호출 
public static returnType functionName(paramType paramName) {
    // body
    return value;
}
Java에서 함수 선언 순서는 무관. 어디서든 호출 가능 (C와 다름).

return 키워드

return: 함수를 즉시 종료하고 값을 반환.
void 메서드에서 return;만 쓰면 조기 종료 가능.
