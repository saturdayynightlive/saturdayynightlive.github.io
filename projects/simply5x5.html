Overview

Simply5x5 is an ad-free iOS workout tracker for the 5x5 strength-training routine. I built it as a solo project to keep the workout flow simple: record sets, adjust the next workout automatically, review progress, and get concise feedback without advertising interruptions.

The app is available on the App Store and was selected for the 2026-1 Computer Programming Extra Project showcase.

View on the App Store · CV

1. Features

  • Ad-free 5x5 workout tracking for the main barbell lifts.
  • Automatic progression and deload logic based on workout success or repeated failure.
  • AI feedback generated from workout-session data through Firebase AI Logic and Gemini.
  • e1RM-based strength ranking using estimated one-rep max values.
  • Plate calculator for quickly loading target barbell weights.
  • HealthKit integration for saving strength-training records to Apple Health.
  • Swift Charts views for reviewing training progress.

2. Implementation Notes

Role Solo developer: product planning, UI implementation, workout logic, data model, and integrations.
App layer SwiftUI screens for onboarding, workout logging, history, rankings, charts, and auxiliary tools.
Progression Successful sessions increase the next target weight; repeated failures trigger a deload.
AI feedback Workout logs are summarized into a prompt and sent through Firebase AI Logic with Gemini.
System APIs HealthKit stores completed workouts, while Swift Charts presents lift trends over time.
